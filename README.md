# ReactJS Example: Simple Booklist App

This project was bootstrapped with the [Create React App](https://github.com/facebookincubator/create-react-app) library.

https://www.npmjs.com/package/create-react-app

For more information about Create React App, review the following:
[Using Create-React-App](CREATE-REACT-APP.md)

This example demonstrates the following:

- React JS Components
- Dumb Components
- Smart Components (Containers)
- React state
- Redux Reducers to extract information from state
- ActionCreators and Actions (to handle events and create custom event-like objects to pass data between components)
